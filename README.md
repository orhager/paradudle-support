# Paradudle Support

**Paradudle** is a practice metronome for drummers — built for focus, feel, and flow.

## Get Help

If you have questions, feedback, or run into any issues, please [open an issue](https://github.com/orhager/paradudle-support/issues/new) and we'll get back to you.

## Features

- Living-color metronome that responds to tempo
- Practice session tracking with progress grid
- Speed training with tempo automation
- Lock Screen controls and Live Activity
- Session timeline with notes and tags

## Privacy

Paradudle does not collect, store, or share any personal data. All practice data stays on your device.
